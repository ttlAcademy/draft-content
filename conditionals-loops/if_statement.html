Decision making is one of the important topics in software flow control. Some codes should be executed only when a particular condition or conditions are met. One of the statements used in Python is if-else block.

If Statement Block

First, let's start with a single if statement. Unless we tell the application, our code normally flows from left to right and top to bottom. During the flow, we want our code whether to choose or not a certain direction. The form of if is presented below.

if expression:
    statement

please pay attention colon : after if expression. It shows us this is a block and after this colon following code belongs to this block. Then notice the indentation. Statement inside if block starts with indentation (normally 4 space 1 indentation). if we draw if statement as a flow chart, it would be:

For example: Imagine we visit a cinema and want to buy a ticket for a film. Cinema has a policy. Only over 15 years old can enter all films without restriction.

your_age = 14
if your_age > 15:
    print('you can enter all films')
print('Rest of the code')

and when you run the program output will be:

Rest of the code

Because your_age variable is less than 15, our if expression returns False and our code inside if block doesn't work. We cannot see print('you can enter all films') output on our screen. Now let's change the variable value from 14 to 18

your_age = 18
if your_age > 15:
    print('you can enter all films')
print('Rest of the code')

Now we can see the code inside if block executed.

you can enter all films
Rest of the code

If-else Statement Block

Our previous example our little cinema program only execute when the condition is True. But what if we want our program to do something else when the condition isn't satisfied? Python has else statement. When the if statement is False, then the code inside else is executed. The form of if - else is presented below.

if expression:
    statement for if
else:
    statement for else

and if - else statement as a flow would be:

Improving our cinema example: According to cinema policy, we also print "you can not enter PEGI 16+ films" to the screen if your_age variable is not greater 15.

your_age = 18
if your_age > 15:
    print('you can enter all films')
else:
    print('you cannot enter PEGI 16+ films')
print('Rest of the code')

and when you run the program output will be:

you can enter all films
Rest of the code

Now write this code to your editor and change the variable value from 18 to 10. Run code and see else statement is executed.
