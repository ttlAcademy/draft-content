Learning Objective ALTA-E-2.1.1 (K4) Analyze a system under test to determine the appropriate automation solution.
Time 60 Minutes
Version 1.1

Read the description of the system. According to the given system and conditions;

Analyze SUT for test automation requirements. Looks amongst others, for the following factors:

  • SUT interfaces
  • Third-party software
  • Level of intrusion
  • Different SUT architectures
  • Size and complexity of SUT
Create a requirements matrix for test automation tools.
  1. Make 3 columns such as Test Tool Requirements, Priority, Explanation.
  2. List requirements on the first column.
  3. On the second column add weight for each requirement according to MoSCoW principle:
    • Must
    • Should
    • Could
    • Won't
  4. Add explanation for the requirement if it has any (optional).
