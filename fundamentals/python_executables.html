So far we have used Python interpreter (command line or Python Shell) to execute our codes. As we progress and learn new things it is obvious we need to write and execute multiline codes and for that point, the Python Executable files with extensions of ".py" come in handy.

Till this lesson, you might have used Python Interpreter. You should notice Python Interpreter is not a convenient way of coding. That is why we are using the code editor. You may use any code editor you wish. I am going to use VS Code which is one of the coolest Open Source code editors in the market, it can be used as a cross-platform in any OS and it is easy to use for beginners.

The coolest thing about code editors: They do a lot of stuff for us like auto-completion (code IntelliSense), sample-ready to use code (boilerplate), color the keywords, etc! In ttl.academy we colored some of the code snippets for your convenience but in real Python console, everything was the same color; When you use the code editors you should see different colors. This is called "syntax highlighting", and it's really useful for code readability. Each different object such as function, strings, comments has different colors so this will allow you to read, track and debug your code, and if something goes wrong you get a clue because of the warning message from editors such as unclosed strings or a typo in a keyword name. All in all coding with editors is very useful and efficient.

The best way to keep our code files for Python is to store them in a separate folder per each project. So create a folder in your PC for your project and give it a name for example "my_project". If you use Windows you may use normal file explorer and create a new folder under a drive for example "C:\my_project"

or you may use "Windows PowerShell" to create a new folder

PS C:\> mkdir my_project
PS C:\> cd my_project

In Linux, you may create your project file under your home directory. First, go to your home directory using the tilde character and then create your directory.

your_user@yourPC:/$ cd ~
your_user@yourPC:~$ mkdir my_project

and go to your new directory.

your_user@yourPC:~$ cd my_project

If you haven't installed VS Code go to VS Code website and install it. After installing your editor you need to install Python extension for linting and code intelliSense. Open your VS Code. You may write "code ." to your command line. This works both for LinuxShell and PowerShell.

your_user@yourPC:~/my_project$ code .
Alternately, you can run VS Code through the operating system UI, then use File > Open Folder to open the project folder.

By starting VS Code in a folder, that folder becomes your "workspace"

Create Your First Python Executable File

From the File Explorer toolbar, select the New File button on the hello folder:

Name the file giving it a descriptive name like <descriptive_name.py> and it automatically opens in the editor. Let's name the file write_my_full_name.py and save it. We can name the file whatever we want just to make sure the file ends in .py. The .py extension tells our Python (if it is installed in our OS) that this is a Python executable file and Python can run it.

Now that you have a code file in your Workspace, enter the following source code in write_my_full_name.py:

first_name = input("Please enter first name: ")
last_name = input("Please enter surname name: ")
my_fullname = f"Your ful name is: {first_name} {last_name}"
print (my_fullname)
my_upper_case_name = my_fullname.upper()
print (my_upper_case_name)

You should be familiar with this code since so far we have covered these topics except line 3. On line 3, we are using a new way of string formatting for Python. It is similar to normal string but we use the "f" letter before the quotation. We can use our variables between open and close curly braces "{}". That is a place holder for any Python object inside a string. When you start typing your code, you may notice how VS Code IntelliSense offers auto-completion options.

Run write _full_name.py

It's very easy to run our code with Python. Just click the "Run Python File in Terminal" button located on the top-right side of the editor.

When you run your code, VS Code automatically save your file and open the terminal panel. The terminal panel will execute the write_my_full_name.py. The python command will run for Linux/macOS: "python3 hello.py" and for Windows: "python write_my_full_name.py.

Congragulations you have written your first Python Executable files with VS Code.
