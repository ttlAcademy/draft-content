For Loop

For loop is one of iteration method in python that is used to repeat blocks of code.

Below is the basic syntax of for loop:

for var in sequence:
    statement(s)

Var is the variable that is used temporarily as a container to store the value of the sequence when the loops occur. The sequence is a type of sequential data such as string, list and tuple. The loops occur until it reaches the last element or list of sequence.


Here is the flow diagram of for:



To loop over a list or any other collection, you can see the following example:

number = [1,2,3,4,5]
for x in number:
    print(x)

The number collection contains five numbers. The loop will iterate over the collection and store each number in x variable and then execute the statement to print each number in the console.

When you run the code above, the output will be:

1
2
3
4
5

Range() Function

You can also use for with range function to give you a simple way to list numbers following one after the other as the example below:

for x in range (1,5):
    print ("This is list number", x)

The output will be:

This is list number 1
This is list number 2
This is list number 3
This is list number 4

You need to note that the range is counted from 1 to 4 and number 5 is not included in the list that is output by python. It means that it includes the first value, but not the last value.
