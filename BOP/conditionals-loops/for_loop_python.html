For Loop

For loop is one of iteration method in python that is used to repeat blocks of code.

Below is the basic syntax of for loop:

for var in sequence:
    statement(s)

Var is the variable that is used temporarily as a container to store the value of the sequence when the loops occur. The sequence is a type of sequential data such as string, list and tuple. The loops occur until it reaches the last element or list of sequence.

Looping over a string

You can do looping over characters of a string using for loop. In python, strings can be indexed like lists. The loop runs as many times as characters that are in the string. Have a look at the code below:

country = 'Belgium'
for i in country:
    print(i)

The output is:

B
e
l
g
i
u
m

Range() Function

If you want to list specific numbers, using range() function is very useful. Range() function gives you a simple way to list numbers, following one after the other. For example, you want to show a list of number from 1 to 10, then you can run the following code:

for i in range (1,11):
    print ('number', i)

When you run the code above, you will see this output:

number 1
number 2
number 3
number 4
number 5
number 6
number 7
number 8
number 9
number 10

The range() function shows values from 1 to 10 and 11 is not part of the output, even though the function input was range(1,11). It means that this function returns the first value, but not the last value.


Break Statement

With the break statement, you can terminate or stop the loop when a condition is triggered. Here is an example:

for i in range (1,11):
    if i == 7:
        break
    print ('number', i)

When you run the program, then the output will be:

number 1
number 2
number 3
number 4
number 5
number 6

To help you understand better, we'll give you another example of this function: 

country = 'Belgium'
for i in country:
    if i == 'g':
        break
    print('Give me :', i)

From the example above, you can see that inside the for loop, there is an if statement that has a condtion that if the variable i is equivalent to the letter 'g', then the loop will break.

What will the output be? Here you go:

Give me : B
Give me : e
Give me : l

Continue Statement

There is also another statement called continue statement that is used to skip over the part of a loop where a condition inside the for loop is triggered. Here is the example of using the continue statement:

country = ‘Belgium’
for i in country:
    if i == 'g':
        continue
    print('Give me:', i )

Running the code above will give you this output:

Give me: B
Give me: e
Give me: l
Give me: i
Give me: u
Give me: m

The letter ‘g’ is not printed because the program hits continue statement when i=g. That’s why the line print('Give me:', i ) isn’t run when i='g', but it starts back up at the start of the loop with the next letter which is the letter ‘i’
