While Loop

Another iteration method used in python is while loop. Using the while loop is the most effective way when you don’t know the number of iterations. This function executes a part of the code until the given condition returns false.


The basic syntax of while loop:

while expression:
    statement(s)

statement(s) may be a single statement or a block of statements. The condition may be any expression and true is any non-zero value. The loop iterates while the condition is true. When the condition becomes false, program control passes to the line immediately following the loop.


while loop flow diagram:



Let’s see the given example below to understand it better:

i = 1
while i<=5:
    print("This is while loop example")
    i=i+1

The program first executes the while loop condition. If the condition is true, the program enters the loop and print the output. It continues to print the output until the condition is false. When it is false, the program comes out and stops the loop.


Infinite while loop

Infinite loops occur if a condition never becomes false. It makes the control never comes out of the loop.

Here is an example:

while True:
    print("Stop me if you can")

When you run the code above, you will see the following output infinitely:

Stop me if you can
Stop me if you can
Stop me if you can
Stop me if you can
Stop me if you can
Stop me if you can
...

You can enter CTRL+C to interrupt the execution of the code.


Range() Function

If you want to list specific numbers, using range() function is very useful. Range() function gives you a simple way to list numbers, following one after the other. For example, you want to show a list of number from 1 to 10, then you can run the following code:

for i in range (1,11):
    print ('number', i)

When you run the code above, you will see this output:

number 1
number 2
number 3
number 4
number 5
number 6
number 7
number 8
number 9
number 10

The range() function shows values from 1 to 10 and 11 is not part of the output, even though the function input was range(1,11). It means that this function returns the first value, but not the last value.


Break Statement

With the break statement, you can terminate or stop the loop when a condition is triggered. Here is an example:

for i in range (1,11):
    if i == 7:
        break
    print ('number', i)

When you run the program, then the output will be:

number 1
number 2
number 3
number 4
number 5
number 6

To help you understand better, we'll give you another example of this function: 

country = 'Belgium'
for i in country:
    if i == 'g':
        break
    print('Give me :', i)

From the example above, you can see that inside the for loop, there is an if statement that has a condtion that if the variable i is equivalent to the letter 'g', then the loop will break.

What will the output be? Here you go:

Give me : B
Give me : e
Give me : l

Continue Statement

There is also another statement called continue statement that is used to skip over the part of a loop where a condition inside the for loop is triggered. Here is the example of using the continue statement:

country = ‘Belgium’
for i in country:
    if i == 'g':
        continue
    print('Give me:', i )

Running the code above will give you this output:

Give me: B
Give me: e
Give me: l
Give me: i
Give me: u
Give me: m

The letter ‘g’ is not printed because the program hits continue statement when i=g. That’s why the line print('Give me:', i ) isn’t run when i='g', but it starts back up at the start of the loop with the next letter which is the letter ‘i’
