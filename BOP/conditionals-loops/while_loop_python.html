While Loop

Another iteration method used in python is while loop. It’s like the for loop, used for repeating block of code but the while loop executes the code until the given condition returns false.


The basic syntax of while loop:

while expression:
    statement(s)

statement(s) may be a single statement or a block of statements. The condition may be any expression and true is any non-zero value. The loop iterates while the condition is true. When the condition is false, the program comes out and stops the loop.


while loop flow diagram:



Let’s see the given example below to understand it better:

i = 1
while i<=5:
    print("Hello, Belgium!")
    i=i+1

The program first executes the while loop condition. If the condition is true, the program enters the loop and print the output. It continues to print the output until the condition is false.

The output:

Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!

Infinite while loop

Infinite loops occur if a condition never becomes false. It makes the control never comes out of the loop.

Here is an example:

while True:
    print("Stop me if you can")

When you run the code above, you will see the following output infinitely:

Stop me if you can
Stop me if you can
Stop me if you can
Stop me if you can
...
...
...
Stop me if you can
Stop me if you can
Stop me if you can

You can enter CTRL+C to interrupt the execution of the code.
