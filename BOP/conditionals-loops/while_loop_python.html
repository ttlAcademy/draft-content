While Loop

Another iteration method used in python is while loop. It’s like the for loop, used for repeating block of code but the while loop executes the code until the given condition returns false.


The basic syntax of while loop:

while expression:
    statement(s)

statement(s) may be a single statement or a block of statements. The condition may be any expression and true is any non-zero value. The loop iterates while the condition is true. When the condition is false, the program comes out and stops the loop.


while loop flow diagram:



Let’s see the given example below to understand it better:

i = 1
while i<=5:
    print("Hello, Belgium!")
    i=i+1

The program first executes the while loop condition. If the condition is true, the program enters the loop and print the output. It continues to print the output until the condition is false.

The output:

Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!

Infinite while loop

Infinite loops occur if a condition never becomes false. It makes the control never comes out of the loop.

Here is an example:

while True:
    print("Stop me if you can")

When you run the code above, you will see the following output infinitely:

Stop me if you can
Stop me if you can
Stop me if you can
Stop me if you can
...
...
...
Stop me if you can
Stop me if you can
Stop me if you can

You can enter CTRL+C to interrupt the execution of the code.


The else statement with the while loop

When you use the else statement with a while loop, the else statement is executed when the condition becomes false.

For example:

i = 1
while i<=5:
    print("Hello, Belgium!")
    i+=1
else:
    print("Bye, Belgium!")

When you execute the code above, you will the following result:

Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Bye, Belgium!

The break statement

Using this statement, you can stop the loop even if the condition is true. Let's see an example below:

i = 1
while i<=8:
    print("Hello, Belgium!")
    if i == 5:
        break
    i+=1

The loop is terminated when i is 5 as the following output:

Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!
Hello, Belgium!

The continue statement

With this statement, you can stop the current iteration and continue with the next if the condition is true.

i = 0
while i<5:
    i+=1
    if i == 4:
        continue
    print(i, "Hello, Belgium!")

The output:

1 Hello, Belgium!
2 Hello, Belgium!
3 Hello, Belgium!
5 Hello, Belgium!

In the output, the numbers are printed from 1 to 5, except the number 4 as the number 4 is the condition that needed to be skipped and the loop continues with the next number.
