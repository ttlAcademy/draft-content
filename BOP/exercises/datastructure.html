Exercise - Order The Numbers

Create a program that reads integers from the user till a blank line is entered. When program read all numbers it should display all of the negative numbers, followed by all of the zeros, followed by all of the positive numbers inside a list. 

# Create an empty list to store entered numbers
numbers = []

# Read value from user
number = input("Please enter an integer - (just enter to exit)")

#As long as user enters the integer keep going;
while number != "":
    num = int(number)
    numbers.append(num)
    #Ask user for the next number
    number = input("Please enter an integer - (just enter to exit)")

numbers.sort()
print(numbers)
