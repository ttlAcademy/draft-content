Dictionaries

A Dictionary is another data structure in python and it is a key/value pairs collection. The order can't be guaranteed in dictionary and it is mutable.The curly braces {} is used to create a dictionary.

The basic syntax of a Dictionary:

my_dictionary = {key1: 'value1', key2: 'value2', ..., keyN:'valueN'}

To help you understand better, letâ€™s have a look at the following examples:

Empty Dictionary

empty_dic = {}

Integer key set

countries = {1: 'Belgium', 2: 'Germany', 3:'Netherland', 4:'France'}
print(countries)

The output:

{1: 'Belgium', 2: 'Germany', 3: 'Netherland', 4: 'France'}

Mixed key set

mixed_dic = {1: 'Automation', 'program': 'Python', 'numbers':[1,2,3]}

Accessing elements

Instead of index, dictionaries use keys to access values. We can use keys inside square brackets [] or get() method.

Here are some examples:

that_man = {'name': 'KodMan', 'birth': 1978, 'hobbies': ['tennis', 'ski']}
print(that_man['name'])
print(that_man['hobbies'])
print(that_man['hobbies'][0])

When you run the given code above, the output will be:

KodMan
['tennis', 'ski']
tennis

Adding in and updating a Dictionary

Adding or updating we are using assignment operator '='. If the key already exists, this operation updates the value, in case the key does not exist it will create key/value pairs

# update that_man name
that_man['name'] = 'CodeMan'
print(that_man)

The output:

{'name': 'CodeMan', 'birth': 1978, 'hobbies': ['tennis', 'ski']}

When we add the new key:

# adding 'lastname' into that_man
that_man['lastname'] = 'Byte'
print(that_man)

The output:

{'name': 'CodeMan', 'birth': 1978, 'hobbies': ['tennis', 'ski'], 'lastname': 'Byte'}

Deleting Element in a Dictionary

We are using pop() method to remove an item from a Dictionary.

that_man.pop('lastname')
print(that_man)

The output:

{'name': 'CodeMan', 'birth': 1978, 'hobbies': ['tennis', 'ski']}

Loop through a Dictionary

You can iterate through each item in a Dictionary using for loop. If you use directly item, you get keys. If you use items in the square bracket you get values.

Example:

for i in that_man:
    print('that man:', i)

The output will be all keys:

that man: name
that man: birth
that man: hobbies

If you use iterated keys inside square brackets:

for i in that_man:
    print('that man:', that_man[i])

The output will be all values:

that man: CodeMan
that man: 1978
that man: ['tennis', 'ski']

Delete a Dictionary

You may want to delete your all key/value pairs in the dictionary. In this case you can use clear() method.

that_man.clear()

This method will not delete a dictionary but makes it empty. To delete entire object use del command.

del that_man
