Python Collections

In python, there are four collections of data type which are:

  1. List is an ordered collection of objects, accessed by index and mutable/changable
  2. Tuple is an ordered collection of objects, accessed by index and immutable
  3. Set is an unordered collection of objects, mutable and unindexed
  4. Dictionary is an unordered collection of objects, mutable and indexed

Lists (arrays)

A list is just like the array which is a sequence of values but more versatile. The list objects don’t need to be in the same data type. It can contain one or more items which are enclosed in the square bracket and use comma to separate items.

The basic syntax of the list:

list = [value1, value2, value3, ..., valueN]

To help you understand better, let’s have a look at the following examples:


A list of strings

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries)

The output:

['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']

A list of integers

birthyear= [1985,1986,1988,1989,1999]
print(birthyear)

The output:

[1985, 1986, 1988, 1989, 1999]

A list of mixed datatypes

thatgirl = [28, 'tall', 1.75, 'programmer', True]
print(thatgirl)

The output:

[28, 'tall', 1.75, 'programmer', True]

There are four data type in the mixed list above which are integer (28), string (‘tall’ & ‘programmer’), float (1.75) and True (Boolean)


Accessing items in the list by index

You can access each item in a list by referring to the index number in the square brackets. Each item in a list has an assigned index value. The first item of a list is always at index zero as shown below.

To be more clear, here is an example:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print('I am from', countries[1], 'and I live in', countries[0])

When you run the given code above, the output will be:

I am from Indonesia and I live in Belgium

Negative indexing

You can also access each item in a list with negative indexing which the index of -1 refers to the last item , -2 to the second last item and so on.

Example:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print('I am from', countries[-4], 'and I live in', countries[-5])

The output:

I am from Indonesia and I live in Belgium

Slice lists

In python, you can access a range of items in a list by using the slicing operator.

Example:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries[1:3])

Here is the output:

['Indonesia', 'Turkey']

An example of slicing items from a predefined point to end:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries[3:])

The output:

['Netherland', 'Japan']

An example of slicing items from beginning to a predefined point:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries[:3])

The output:

['Belgium', 'Indonesia', 'Turkey']

If you want to start the search from the end of the list, then you can use negative indexes.

You can have a look at the following example:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries[-4:-1])

The output:

['Indonesia', 'Turkey', 'Netherland']

The example above returns the items from index -4 (included) to index -1 (excluded).


If you want to reverse the items in the list, see the example below:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
print(countries[::-1])

The output:

['Japan', 'Netherland', 'Turkey', 'Indonesia', 'Belgium']

The slice statement [::-1] means start at the end of the string and end at position 0, move with the step -1, the negative one, which means one step backwards.


Add item(s) to a list

Using the insert() method to add an item at the specified index:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries.insert(2,'Spain')
print(countries)

The output:

['Belgium', 'Indonesia', 'Spain', 'Turkey', 'Netherland', 'Japan']

Using append() method to append an element to the end of the list:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
continents= ['Asia','Africa', 'Europe']
countries.append(continents)
print(countries)

The output:

['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan', ['Asia', 'Africa', 'Europe']]

Using the extend() method to add all elements of a list to another list:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries1= ['Singapore','Australia']
countries.extend(countries1)
print(countries)

The output:

['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan', 'Singapore', 'Australia']

Remove item(s) from list

There are some methods to remove item(s) from a list, such as these examples below:


Using remove() method:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries.remove('Japan')
print(countries)

The output:

['Belgium', 'Indonesia', 'Turkey', 'Netherland']

Using pop() method:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries.pop(3)
print(countries)

The output:

['Belgium', 'Indonesia', 'Turkey', 'Japan']

The pop() method removes the specified index. If the index is not specified, it removes the last item.


Using del keyword removes the specified index:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
del countries[1]
print(countries)

The output:

['Belgium', 'Turkey', 'Netherland', 'Japan']

Using clear() keyword removes all the elements from a list:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries.clear()
print(countries)

The output:

[]

Change item value

As we mentioned that list is mutable or changeable. It means that you can change items in the list using the assignment operator (=), as examples below:

Change the 2nd item:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries[1] = 'France'
print(countries)

The output:

['Belgium', 'France', 'Turkey', 'Netherland', 'Japan']

Change 2nd to 4th items:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries[1:4] = ['France','Spain','Germany']
print(countries)

The output:

['Belgium', 'France', 'Spain', 'Germany', 'Japan']

Loop through a list

You can iterate through each item in a list using for loop.

Example:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
for i in countries:
    print('I have been to', i)

The output:

I have been to Belgium
I have been to Indonesia
I have been to Turkey
I have been to Netherland
I have been to Japan

Sort item in a list

Using the sort() method to sort the list ascending by default:

countries= ['Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan']
countries.sort()
print(countries)

The output:

['Belgium', 'Indonesia', 'Japan', 'Netherland', 'Turkey']
