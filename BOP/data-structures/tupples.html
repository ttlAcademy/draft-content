Tuples

A tuple is very similar to a lis it allows us to store some data and repetition of items are also possible.

Only one main difference is a tuple is immutable. What does it mean? You can not add an item to a tuple or you can not update one of the items after you create it. Consider using tuples if you have different types of data in your collection or if your collection needs to be write protected on the other hand consider using lists if you have same types of data in your collection. This is not must have but general best practices.

The basic syntax of a Tuple:

my_tuple = (value1, value2, value3, ..., valueN)

To help you understand better, let’s have a look at the following examples:

Empty Tuple

empty_tuple = ()

A Tuple of strings

countries= ('Belgium', 'Germany', 'France', 'Netherland', 'Poland')
print(countries)

The output:

('Belgium', 'Germany', 'France', 'Netherland', 'Poland')

A Tuple of integers

birthyears= (1985,1986,1988,1989,1999)
print(birthyears)

The output:

(1985, 1986, 1988, 1989, 1999)

A Tuple of mixed datatypes

We can have different types of data inside tuples. Look at example below each item is different type of data 

that_man = (42, 'smart', 1.70, True, ['tennis', 'walking', 'ski'])
print(that_man)

The output:

(42, 'smart', 1.70, True, ['tennis', 'walking', 'ski'])

There are four data types and one data structure in the mixed tuple above which are integer (28), string (‘developer’), float (1.70), True (Boolean) and one list ['tennis', 'walking', 'ski']

Accessing items in the Tuple by index

You can access each item in a tuple by referring to the index number in the square brackets. Each item in a tuple has an assigned index value. When you start from left to right (LTR), the first item of a tuple is always at index zero as we learned during topic lists.

To be more clear, here is an example:

countries= ('Belgium', 'Poland', 'France', 'Netherland', 'Germany')
print('I am from', countries[1], 'and I live in', countries[0])

When you run the given code above, the output will be:

I am from Poland and I live in Belgium

Negative indexing

You can also access each item in a tuple with negative indexing which the index of -1 refers to the last item , -2 to the second last item and so on. In this case you are starting to count from right to left (RTL). Again Indexing logic is the same as in the lists.

Example:

countries = ('Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan')
print('I am from', countries[-3], 'and I live in', countries[-5])

The output:

I am from Turkey and I live in Belgium

Slice Tuples

In python, you can access a range of items in a tuple by using the slicing operator.

Example:

countries= ('Belgium', 'Germany', 'Poland', 'Netherland', 'Japan')
print(countries[1:3])

Here is the output:

('Germany', 'Poland')

An example of slicing items from a particular point to end:

countries= ('Belgium', 'Germany', 'Poland', 'Netherland', 'Japan')
print(countries[3:])

The output:

('Netherland', 'Japan')

An example of slicing items from beginning to a particular point:

countries= ('Belgium', 'Germany', 'Poland', 'Netherland', 'Japan')
print(countries[:3])

The output:

('Belgium', 'Germany', 'Poland')

If you want to start the search from the end of the tuple, then you can use negative indexes.

You can have a look at the following example:

countries= ('Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan')
print(countries[-4:-1])

The output:

('Indonesia', 'Turkey', 'Netherland')

The example above returns the items from index -4 (included) to index -1 (excluded).


If you want to reverse the items in the tuple, see the example below:

countries= ('Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan')
print(countries[::-1])

The output:

('Japan', 'Netherland', 'Turkey', 'Indonesia', 'Belgium')

The slice statement [::-1] means start at the end of the string and end at position 0, move with the step -1, the negative one, which means one step backwards.

Is it possible to change a Tuple?

As we mentioned earlier tuples are immutable. An item of a tuple ca not be changed once it is assigned. If you try to change an item of a tuple;

countries[2] = 'Germany'

you will get type error.

Traceback (most recent call last):
    File "<pyshell#23>", line 1, in <module>
        countries[2] = 'Germany'
TypeError: 'tuple' object does not support item assignment

But there are some tricks and details need to be mentioned.

  • You can reassign tuple, in this case you can change as a whole object.

    countries= ('Spain', 'Sweden', 'Italy', 'Denmark', 'Norway')

  • If item inside a tuple is mutable, this item can be changed.

    that_man = (42, 'smart', 1.70, True, ['tennis', 'walking', 'ski'])
that_man[4][0] = 'football'
print(that_man)

    The output:

    (42, 'smart', 1.70, True, ['football', 'walking', 'ski'])

    Since the last item is type of list, python accepts this changes. Because we are changing an item of the list.

  • You can convert a tuple into a list, change the item whatever you want and then you can convert it back to tuple.

    that_man = (42, 'smart', 1.70, True, ['tennis', 'walking', 'ski'])
that_man_list = list(that_man)
that_man_list[1] = 'dummy'
that_man = tuple(that_man_list)
print(that_man)

    The output:

    (42, 'dummy', 1.70, True, ['tennis', 'walking', 'ski'])

    We have used list() and tuple() functions for conversion between two types.

Using Boolean Logic in a Tuple

We can check any item if it exist in a tuple or not. This boolean operation return True or False according to operation result. To achieve this we use in keyword.

countries= ('Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan')
check_result = 'Belgium' in countries
print(check_result)
check_result = 'Spain' in countries
print(check_result)

The output:

True
False

Loop through a Tuple

You can iterate through each item in a Tuple using for loop.

Example:

countries= ('Belgium', 'Indonesia', 'Turkey', 'Netherland', 'Japan')
for i in countries:
    print('I have been to', i)

The output:

I have been to Belgium
I have been to Indonesia
I have been to Turkey
I have been to Netherland
I have been to Japan

Delete a Tuple

You may want to delete your tuple. In this case you can use del command. You need to be sure when you use this command. It deletes entire tuple. 
del countries
