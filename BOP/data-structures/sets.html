Sets

Set is another data structure in Python. It does not allow us to enter same value as an item. So no repetition. The another key point is Sets never guarantee the order of its items, so it is unordered.

The basic syntax of a Set:

my_set = {value1, value2, value3, ..., valueN}

To help you understand better, letâ€™s have a look at the following examples:

Empty Set

As you se we are using curly braces {} for set. But another structure, dictionaries which we learn on the next lesson also use curly braces. To create empty set we should use set() function. Otherwise my_collection = {} will create empty dictionary for us.

empty_set = set()

A Set of strings

countries= {'Belgium', 'Germany', 'France', 'Netherland', 'Poland'}
print(countries)

The output:

{'Belgium', 'Germany', 'France', 'Netherland', 'Poland'}

A Set of integers

birthyears= {1985,1986,1988,1989,1999}
print(birthyears)

The output:

{1985, 1986, 1988, 1989, 1999}

A Set of mixed datatypes

We can have different types of data inside Sets. Look at example below each item is different type of data. Only primitive types and immutable types are allowed. You cannot put a list inside a set. We will cover immutability on the following lines. 

that_man = {42, 'smart', 1.70, True, ('tennis', 'walking')}
print(that_man)

The output:

{42, 'smart', 1.70, True, ('tennis', 'walking'))

There are four data types and one data structure in the mixed set above which are integer (42), string (smart), float (1.70), Boolean (True) and tuple ('tennis', 'walking')

Set is Semi-mutable

Unlike tuple we can add and remove an item to/from a Set. But we can not change item itself. So items are immutable inside Set whereas Set is mutable. Let's have below example.

Add item to a Set

birthyears= {1985,1986,1988,1989,1999}
birthyears.add(2000)
print(birthyears)

The output:

{1985,1986,1988,1989,1999,2000}

when you try to add an item already inside the Set you will not get an error, but adding same item will not responded and Set will not change

birthyears.add(2000)
print(birthyears)

The output will be the same as it is:

{1985,1986,1988,1989,1999,2000}

Remove item from a Set

birthyears= {1985,1986,1988,1989,1999}
birthyears.remove(1986)
print(birthyears)

The output:

{1985,1988,1989,1999,2000}

Updating a Set

update() method can be used to add multiple items at once. In this case pass the multiple items inside a list.

birthyears= {1985,1986,1988,1989,1999}
birthyears.update([2001,2002,2003,2004])
print(birthyears)

The output:

{1985,1988,1989,1999,2000,2001,2002,2003,2004}

Mutable items are not allowed

We cannot pass mutable item into a Set. Below here [2005, 2006] is a list which is mutable as we have learned. If you try this you will get an error.

birthyears= {2001,2002,2003,[2005,2006]}

The output:

Traceback (most recent call last):
    File "<pyshell#90>", line 1, in <module>
        birthyears= {2001,2002,2003,[2005,2006]}
TypeError: unhashable type: 'list'

Accessing an item by index is not possbile

If you ever try to access any item by index you will get an error.

print(birthyears[1])

The output:

Traceback (most recent call last):
    File "<pyshell#90>", line 1, in <module>
        print(birthyears[1])
TypeError: 'set' object is not subscriptable

Loop through a Set

You can iterate through each item in a Set using for loop.

Example:

for i in birthyears:
    print(i)

The output:

1985
1988
1989
1999
2000
2001
2002
2003
2004

Clearing a Set

clear() method makes empty set.

birthyears.clear()
print(birthyears)

The output:

set()
