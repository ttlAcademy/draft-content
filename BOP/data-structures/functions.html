Functions

Functions are blocks of codes that are designed to perform a particular task/tasks. You define these tasks in a function. Function does not run unless you execute them. The common way to run function is call the function name.

The basic syntax of a Function:

def my_function(0 or more arguments):
    Do something
    return a value # (optional)

#then call the function to execute
my_function()

As you see, we are using def key word to define a function and give it a name. Here is my_function. Then parentheses are used. After first line function scope stars with indentation. To help you understand better, letâ€™s have a look at the following examples:

This example of a function named my_first_function() that only prints a text "Hi! This is my first function"

def my_first_function()
    print("Hi! This is my first function")
    
my_first_function()

The output:

Hi! This is my first function

Function Scope

When you declare an object for example a variable, it will be accessible only inside this function and outside of function it will not be exist

def write_my_city():
    my_city = 'Brussels'
    print('My city is', my_city)

write_my_city()

The output:

My city is Brussels

Here is my_city variable is only valid and exist inside print_my_city function. If you try to access my_city variable outside of the function, you will get error.

Using "return" keyword

We are using return keyword to return a value from function. Let's look at below example

def return_my_city():
    return 'Brussels'

my_city = return_my_city()
print(my_city)

Here is our function returning "Brussels" and we assign this returning value to my_city variable

The output:

Brussels

Parameters

You can define parameters between the parentheses. Function may have 0 or more parameters. If we use more than one parameter then we use comma to seperate them.

def calculate_square(number):
    return number * number

square_result = calculate_square(5)
print(square_result)

The output will be:

25

Before giving last example it is nice to remember programming terminology here again. When we are defining function we use number variable and this is named parameter. When we call the function we pass number 5 to the function this is argument.So we can say when you call the function with parameters you should pass arguments according parameters used.

Example of two parameters

def write_full_name(first_name, last_name):
    full_name = first_name + ' ' + last_name
    print(full_name)
    
write_full_name('John', 'Doe')

The output:

John Doe
